# @wanglihua/create-ezreal-app

React脚手架

## 使用

当前使用node版本：v14.17.5
推荐使用yarn
以下命令以yarn给出，npm同理

##### 安装
```bash
yarn 
```


##### 启动
```bash
yarn start
```

##### 构建
```bash
yarn build
```

##### eslint检查
```bash
yarn lint:js
yarn lint:style
yarn lint:prettier
yarn lint
```

##### eslint修复
```bash
yarn lint:fix
```

##### 格式化代码
```bash
yarn prettier
```

##### 提交代码
```bash
yarn commit
```

##### 版本发布
```bash
yarn release
```
